## :information_source: Run the project

```bash
# Clone the repo
$ git clone https://github.com/Svaity/Whatsapp-clone-MERN.git

# Get into the directory
$ cd Whatsapp-clone-MERN-master

#Install the dependancies
$ yarn  install

#Run the server
$ yarn start

```
